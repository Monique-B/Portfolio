Project web

Project web Ontwerp en maak een online interactieve reisgids waarmee je je reis kan plannen. Maak een prototype dat toont wat je voor, tijdens en na de reis met de website kan doen. Laat zien wat voor informatie er te vinden is, en misschien kun je er ook iets achterlaten voor andere reizigers. Het wordt een seamless multi-device experience.

Wat heb ik gedaan

MeetUp is een interactieve reisgids voor backpackers. Ze kunnen deze multi-device applicatie gebruiken voor, tijdens en na hun reis. Veel backpackers gaan alleen op reis, maar vinden het wel leuk om andere mensen te ontmoeten en sommige delen samen te kunnen reizen. Met MeetUp kun je elke dag zien wie waar bij welke bezienswaardigheid is die jij wilt bezoeken. De app biedt veel informatie over de locatie die jij wilt gaan bezoeken. Je kunt jezelf dus goed inlezen voordat je op reis gaat.

Visual Interface Design

Visual Interface Design De visual interface designer is iemand die verantwoordelijk is voor de layout van de interface tot in elk detail. De visual interface designer ontwerpt de schermen waarmee de gebruiker interacteert. Een optimale visuele gebruikerservaring is het doel. Waar staat welke informatie in het scherm. Ook is de visual interface designer verantwoordelijk voor consistentie van de visuele taal, die het handelen van de gebruiker bepaalt. Daarnaast weet de visual interface designer ook een coherente stijlgids te maken.
